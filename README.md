# reactlibx

## 介绍 / Intro

react的一些组件 。


## 安装 / Install

```
npm i reactlibx
```

### import 整个包 

```javascript
import * as reactlibx from 'reactlibx'
```

### import 单个组件

```javascript
import {PageAutoSizeWrapper} from 'reactlibx'
```
